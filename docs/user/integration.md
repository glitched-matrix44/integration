# Integration Module

## What is the Integration Module?

The Integration module lets you connect your application to third-party platforms such as Zoho Books, WooCommerce, Gautam's Bot, and Entity services. Once connected, data from these platforms is mapped, synced, and stored locally.

## Supported Integrations

| Provider | Type | Description |
|----------|------|-------------|
| Zoho Books | Accounting | Sync financial and invoice data |
| WooCommerce | E-commerce | Sync products, orders, and customers |
| Gautam's Bot | Automation | Bot-based workflow and messaging automation |
| Entity | Data Service | Sync and manage entity-level data records |

## How to Add an Integration

1. Navigate to **Integrations** from the main menu
2. Click **Add Integration**
3. Select your provider from the supported list
4. Fill in the required configuration fields (API keys, secrets, etc.)
5. Set the status to **Active** and save

## How to Sync Data

Once an integration is active, use the sync option from the integration detail page to pull the latest data from the provider.

## Managing Integrations

- **Edit** — Update credentials or configuration at any time
- **Deactivate** — Set status to inactive to pause syncing without deleting
- **Delete** — Permanently removes the integration and its configuration
